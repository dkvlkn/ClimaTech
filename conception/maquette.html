Prévisions météo

Carte interactive de France 🗺️ + menus déroulants région ➝ département ➝ ville

Affichage des données météo : température, condition, prévisions à 7 jours.

Statistiques 📈

  • Histogramme (consultations par région)
  • Camembert (répartition des requêtes)
  • Courbe (évolution du trafic)
  • Top 5 des villes les plus consultées
  • Bouton “Tendances météo”

Informations techniques ⚙️

Détails des APIs utilisées :

  • OpenWeatherMap : météo en JSON
  • NASA APOD : image ou vidéo du jour
  • Géolocalisation : GeoPlugin, ipinfo, whatismyip
  • Stockage local : CSV, cache JSON/XML

Évolutions prévues 🚀

  • ⚠️ Affichage des alertes météo (vent, orages, canicule)
  • 🧾 Export PDF de la météo hebdomadaire
  • 🌅 Fond d’écran dynamique selon la météo actuelle
  • 📱 Version mobile optimisée avec menu “burger” responsive
  • 📴 Mode hors-ligne (PWA - Progressive Web App)
  • 📊 Statistiques détaillées (par ville, date, navigateur)
  • 📈 Graphiques météo (jours les plus chauds, pluvieux...)
  • 🏆 Classement des villes les plus consultées
